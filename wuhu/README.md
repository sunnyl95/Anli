# 芜湖税务局发票虚开预测案例

## 案例任务背景

**案例任务是基于于企业发票虚开事实行为特征，根据历史案发数据标识，使用模型算法提取虚开发票特征，解决识别发票虚开的难题。<p>
该问题预测的标签为：某企业或零售商是否存在发票虚开的情况，发票虚开标签为1，发票未虚开标签为0。<p>
该案例中，用于评价模型的评价指标为指定的打分标准，且该指标越大效果越好。<p>
